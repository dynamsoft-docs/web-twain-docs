# UI Customization

## Is UI of the Dynamic Web TWAIN SDK fully customizable? What can’t be customized?

The UI of dynamic web TWAIN SDK consists of the viewer, thumbnail viewer, image editor among other html UI elements. The display language of the scan page can be customized, the customer can surely create a scan page in their local language as the interface is all about HTML element. Also, the built-in errors can be translated to their own language. Progress bar, image editor can also be customized. All the parts are fully customizable except scanner UI, which is not customizable.
